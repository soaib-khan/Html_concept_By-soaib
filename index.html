HTML Basic to Advanced Course

1. Basic Structure

<!DOCTYPE html>
<html>
  <head>
    <title>My Website</title>
  </head>
  <body>
    Content goes here
  </body>
</html>

Explanation: This is the basic HTML structure. It contains the <head> for metadata and <body> for visible content.

2. Headings

<h1>Heading 1</h1>
<h2>Heading 2</h2>
...
<h6>Heading 6</h6>

Explanation: Headings are used for titles and sub-titles. <h1> is the largest and <h6> is the smallest.

3. Paragraph and Line Break

<p>This is a paragraph.</p>
<br>  
<hr>

Usage: For writing content and separating sections.

4. Text Formatting Tags

<b>Bold</b>
<i>Italic</i>
<u>Underline</u>
<mark>Highlight</mark>
<small>Small text</small>
<del>Deleted</del>
<ins>Inserted</ins>

Usage: To emphasize or style text.

5. Lists

<ul>
<li>Unordered item</li>
</ul>

<ol>
<li>Ordered item</li>
</ol>

Explanation: <ul> for bullet points, <ol> for numbered lists.

6. Images

<img src="image.jpg" alt="description" width="200" height="150">

Usage: Display images in your webpage. Use alt for accessibility.

7. Links

<a href="https://example.com" target="_blank">Visit Website</a>

Explanation: Use <a> to create hyperlinks.

8. Tables

<table border="1">
<tr><th>Name</th><th>Age</th></tr>
<tr><td>Soaib</td><td>21</td></tr>
</table>

Usage: Display tabular data using rows (<tr>) and columns (<td>).

9. Forms

<form action="submit.php" method="post">
<input type="text" name="username">
<input type="password" name="password">
<input type="submit">
</form>

Usage: Collect user input via text boxes, buttons, and more.

10. Div and Span

<div>Block level element</div>
<span>Inline element</span>

Usage: <div> for block layout, <span> for inline styling.

11. Semantic Tags

<header></header>
<nav></nav>
<main></main>
<section></section>
<article></article>
<footer></footer>

Usage: Better structure and SEO for your web pages.

12. Audio and Video

<audio controls>
<source src="audio.mp3" type="audio/mpeg">
</audio>

<video width="320" height="240" controls>
<source src="movie.mp4" type="video/mp4">
</video>

Usage: Embed multimedia content on webpages.
